ğŸ’¡ Contoh Program Sederhana

Program berikut meminta pengguna memasukkan dua angka, lalu mencetak hasil penjumlahannya: 


#include <iostream>
using namespace std;

int main() {
    int a, b, hasil;

    cout << "Masukkan angka pertama: ";
    cin >> a;

    cout << "Masukkan angka kedua: ";
    cin >> b;

    hasil = a + b;

    cout << "Hasil penjumlahan: " << hasil << endl;

    return 0;
}
