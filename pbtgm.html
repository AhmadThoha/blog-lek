Dasar Pemrograman C++

C++ adalah bahasa pemrograman tingkat menengah yang banyak digunakan dalam pengembangan perangkat lunak, game, sistem embedded, dan aplikasi desktop. Materi ini mengenalkan siswa pada struktur dan logika dasar dalam membangun program menggunakan C++.

Materi yang Dipelajari:

  • Struktur program C++ dan fungsi main()
  • Deklarasi variabel dan tipe data (int, float, char, dll.)
  • Operator aritmatika dan logika
  • Penggunaan kontrol alur (if, switch, for, while)
  • Pengenalan fungsi dan modularisasi
  • Input/output dasar menggunakan cin dan cout

Contoh Program Sederhana:

Program di bawah ini akan meminta pengguna memasukkan dua angka, lalu menampilkan hasil penjumlahannya: 


#include <iostream>
using namespace std;

int main() {
    int a, b, hasil;
    
    cout << "Masukkan angka pertama: ";
    cin >> a;

    cout << "Masukkan angka kedua: ";
    cin >> b;

    hasil = a + b;

    cout << "Hasil penjumlahan: " << hasil << endl;

    return 0;
}

Program di atas menggunakan cin untuk input, dan cout untuk output. Struktur dasar seperti ini sangat penting untuk pemula sebelum melanjutkan ke konsep lebih kompleks seperti array, function, dan OOP.
