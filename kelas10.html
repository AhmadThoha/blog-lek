📚 Materi Dasar Pemrograman

HTML, CSS, Python, Java, PHP untuk siswa kelas 10 SMK

📘 HTML (HyperText Markup Language)

HTML digunakan untuk membuat struktur halaman web.

<h1>Halo Dunia</h1>
<p>Ini paragraf pertama saya.</p>

🔍 Menampilkan judul dan paragraf. Elemen dasar untuk konten web.

🎨 CSS (Cascading Style Sheets)

CSS digunakan untuk mengatur tampilan HTML.

h1 {
  color: blue;
  font-size: 24px;
}

🔍 Mengubah warna dan ukuran teks <h1>.

🐍 Python

Bahasa yang mudah dibaca dan cocok untuk pemula.

nama = input("Siapa namamu? ")
print("Halo, " + nama)

🔍 Menyapa pengguna berdasarkan input yang dimasukkan.

☕ Java

Bahasa OOP yang banyak digunakan di aplikasi Android dan desktop.

public class Halo {
  public static void main(String[] args) {
    System.out.println("Halo Dunia!");
  }
}

🔍 Program sederhana untuk mencetak "Halo Dunia!"

🐘 PHP

Bahasa server-side populer untuk website dinamis.

<?php
$nama = "Andi";
echo "Halo, $nama!";
?>

🔍 Menyimpan data dalam variabel dan menampilkannya.

Dengan memahami kelima dasar ini, kamu siap lanjut belajar JavaScript, framework web, dan pengembangan aplikasi nyata!
