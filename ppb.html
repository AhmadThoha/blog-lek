PPB (Pemrograman Perangkat Bergerak)

Pada mata pelajaran ini, siswa akan belajar membangun aplikasi mobile berbasis Android dan iOS menggunakan Flutter, framework UI modern yang menggunakan bahasa Dart. Fokus pembelajaran mencakup pembuatan antarmuka, manajemen state, hingga penggunaan database lokal.

Materi Utama:

  • Pengenalan Flutter dan bahasa Dart
  • Struktur project Flutter dan penggunaan widget
  • Layouting responsif dengan Row, Column, Container, ListView
  • Handling event dan interaktivitas (onPressed, gesture)
  • Navigasi antar halaman dengan Navigator
  • Penggunaan state management dasar (setState)
  • Database lokal dengan Hive atau SQLite
  • Build dan deploy ke perangkat Android/iOS

Contoh Kode Flutter (Hello World):


import 'package:flutter/material.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: Text('Hello Flutter')),
        body: Center(child: Text('Halo Dunia!', style: TextStyle(fontSize: 24))),
      ),
    );
  }
}

Contoh di atas adalah aplikasi Flutter paling dasar. Siswa akan belajar mengembangkan aplikasi lebih kompleks seperti To-Do List, Kalkulator, dan Aplikasi Kasir berbasis Flutter.
