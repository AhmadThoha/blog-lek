🗄️ Materi Basis Data Lanjutan

SQL, JOIN, Subquery, dan DBMS untuk siswa kelas 11 RPL

🧩 DBMS (Database Management System)

DBMS adalah perangkat lunak untuk membuat dan mengelola database. Contohnya: MySQL, Oracle, PostgreSQL, SQL Server.

🔍 DBMS memudahkan penyimpanan, akses, dan manipulasi data secara efisien.

📐 DDL (Data Definition Language)

  • CREATE – Membuat tabel atau objek database.
  • ALTER – Mengubah struktur tabel (menambah/ubah kolom).
  • DROP – Menghapus tabel atau objek.

🔍 DDL digunakan untuk mendefinisikan struktur database.

📝 DML (Data Manipulation Language)

  • SELECT – Mengambil data dari tabel.
  • INSERT – Menambahkan data ke tabel.
  • UPDATE – Mengubah data di tabel.
  • DELETE – Menghapus data dari tabel.

🔍 DML digunakan untuk manipulasi isi data dalam tabel.

🔒 DCL (Data Control Language)

  • GRANT – Memberikan hak akses ke pengguna.
  • REVOKE – Menghapus hak akses dari pengguna.

🔍 Digunakan untuk mengatur hak akses pada database.

🔁 Subquery (Query Bersarang)

Subquery adalah query yang berada di dalam query lain.

SELECT * FROM produk 
WHERE harga > (SELECT AVG(harga) FROM produk);

🔍 Menampilkan produk yang harganya di atas rata-rata.

🔗 JOIN (Menggabungkan Tabel)

JOIN digunakan untuk mengambil data dari beberapa tabel yang saling berelasi.

  • INNER JOIN: Menampilkan data yang cocok di kedua tabel.
  • LEFT JOIN: Menampilkan semua data dari tabel kiri dan yang cocok dari tabel kanan.
  • RIGHT JOIN: Menampilkan semua data dari tabel kanan dan yang cocok dari tabel kiri.
SELECT pelanggan.nama, transaksi.total
FROM pelanggan
INNER JOIN transaksi ON pelanggan.id = transaksi.pelanggan_id;

🔍 Menampilkan nama pelanggan dan total transaksi mereka.
