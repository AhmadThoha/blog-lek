🌐 Pemrograman Web & Keamanan

Front-End, Back-End, dan Dasar Keamanan Web untuk siswa SMK

🎨 Front-End Web

Menggunakan HTML, CSS, dan JavaScript untuk membangun antarmuka pengguna (UI).

<button onclick="alert('Halo!')">Klik Saya</button>

🔍 Tombol interaktif sederhana menggunakan JavaScript.

🔧 Back-End Web

Pemrosesan data di sisi server menggunakan bahasa seperti PHP atau Node.js.

<?php
$nama = "Siswa";
echo "Selamat datang, $nama!";
?>

🔍 Script PHP sederhana untuk menampilkan ucapan selamat datang.

🔗 API (Application Programming Interface)

Menghubungkan antar aplikasi menggunakan REST API atau JSON.

fetch('/api/data')
  .then(response => response.json())
  .then(data => console.log(data));

🔍 Contoh permintaan API menggunakan JavaScript.

🔐 Keamanan Web

  • Session & Cookie: Menyimpan informasi pengguna.
  • Session Hijacking: Ancaman mencuri session pengguna.
  • Validasi Form: Cek input agar tidak membahayakan sistem.
  • Praktik Aman: Gunakan HTTPS, CSRF Token, dan enkripsi password.

🔍 Praktik keamanan penting agar aplikasi tidak mudah diserang.

💻 Contoh Struktur HTML

<!DOCTYPE html>
<html>
<head>
  <title>Contoh HTML</title>
</head>
<body>
  <h1>Halo Dunia!</h1>
  <p>Ini paragraf pertama dalam HTML.</p>
</body>
</html>

🔍 Struktur dasar dokumen HTML.

Dengan memahami pemrograman web dan keamanannya, kamu bisa membuat aplikasi web yang modern, interaktif, dan aman!
