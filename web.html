Dasar Pemrograman Web & Keamanan

Pemrograman web adalah proses membuat aplikasi berbasis browser yang interaktif dan dinamis. Materi ini mencakup pemahaman dasar front-end, back-end, serta keamanan data pengguna.

  • Front-End Web: HTML, CSS, JavaScript untuk membuat tampilan antarmuka pengguna (UI).
  • Back-End Web: Penggunaan PHP, Node.js, atau Python untuk logika server dan pengolahan data.
  • API (Application Programming Interface): Pertukaran data antar sistem, misalnya menggunakan REST API atau JSON.
  • Session & Cookie: Teknik penyimpanan data pengguna agar tetap login atau menyimpan preferensi pengguna.
  • Session Hijacking: Ancaman keamanan di mana peretas mencuri session pengguna. Pencegahan: penggunaan HTTPS, token CSRF, dan logout otomatis.
  • Validasi Form: Mencegah input berbahaya dengan validasi di sisi klien (JavaScript) dan server (PHP, dll).
  • Keamanan Dasar: Sanitasi input, enkripsi password, dan pembatasan akses berdasarkan role pengguna.

Memahami konsep-konsep ini membantu siswa dalam membuat aplikasi web yang aman dan profesional, serta siap terhubung dengan sistem modern seperti database dan layanan pihak ketiga.
